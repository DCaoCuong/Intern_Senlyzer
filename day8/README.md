## API Reference: API Reference: Server/App APIs 

1. Tìm hiểu API xử lý Cache: Tập trung nghiên cứu chi tiết cách sử dụng các hàm tiện ích Server-only như revalidatePath và revalidateTag để thực hiện xác thực lại dữ liệu theo yêu cầu (On-demand Revalidation).

2. Tìm hiểu Request Headers & Cookies: Phân tích các hàm chỉ chạy trên Server như cookies() và headers() để truy cập và thao tác với dữ liệu từ request, hiểu rõ cách chúng được sử dụng trong Server Components/Route Handlers.

3. Thực hành Server Utilities: Code: Xây dựng một ví dụ đơn giản (ví dụ: một Route Handler hoặc Server Action) sử dụng kết hợp các API trên để thay đổi dữ liệu và kích hoạt cơ chế revalidatePath hoặc ghi/đọc cookies.

4. Ôn tập về sự khác biệt giữa App Router và Pages Router. Đặc biệt, tập trung vào cách xử lý phiên bản React: App Router sử dụng React canary releases tích hợp sẵn, trong khi Pages Router sử dụng phiên bản React được cài đặt trong package.json.

5. Hệ thống hóa lại các khái niệm quan trọng đã học (Server Components, Client Components, Suspense, Caching, Server Actions, Route Handlers, Middleware).
