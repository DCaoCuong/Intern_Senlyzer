## Optimization & UI

1. Image Optimization: Tìm hiểu concept, cách sử dụng component <Image> để xử lý việc thay đổi kích thước, định dạng và Lazy Loading hình ảnh tự động.
- Remote    
- Local

2. Thực hành Component <Image>: Áp dụng component này thay cho thẻ <img> truyền thống, kiểm tra kết quả tối ưu hóa (resize, webp/avif) và hiệu ứng lazy loading.

3. Font Optimization: Tìm hiểu về component next/font, cách nó tối ưu hóa phông chữ bằng cách loại bỏ các yêu cầu mạng bên ngoài và tự động lưu trữ, nhằm cải thiện hiệu suất tải trang.
- Local Fonts
- Google Fonts

4. Giới thiệu Metadata API: Tìm hiểu cách quản lý các thông tin Metadata (tiêu đề, mô tả, Open Graph, v.v.) bằng File-based Metadata API trong App Router.

5. Thực hành Metadata tĩnh: Áp dụng metadata object hoặc file để định nghĩa các thông tin SEO tĩnh cho toàn bộ ứng dụng (layout.js) và từng trang cụ thể (page.js).

6. Thực hành Metadata động: Học cách tạo hàm generateMetadata để xử lý các Dynamic Metadata (ví dụ: lấy tiêu đề từ slug của bài viết) dựa trên dữ liệu tìm nạp.
